Proyecto participante en ExpoCiencias 2019 y CIM de Orizaba, que constaba de varias partes:

1; Un sitio web donde se muestra información relevante del daltonismo y los problemas que esta discapacidad provee.
2; Una aplicación móvil la cuál adaptaba el “Test Ishihara” hacia niños menores, mostrando imagenes más fáciles de reconocer para ellos.
3; Una simbologia que podía usarse para identificar los colores primarios, secundarios y terciarios.
4; Aplicación de Realidad Virtual en la que se tenían 2 escenarios mostrando las posibles maneras de usar la simbología.
